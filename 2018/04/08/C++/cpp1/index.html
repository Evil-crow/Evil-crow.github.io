一, 内置类型

C++的内置类型分为算术类型与空类型

内置类型又分为算术类型和浮点型

类型 含义 最小尺寸
bool 布尔类型 未定义
char 字符 8
wchar_t 宽字符 16
char16_t Unicode字符 16
char32_t Unicode字符 32
short 短整型 16
int 整型 16
long 长整型 32
long long 长整型 64
float 单精度 6位有效数字
double 双精度 10位有效数字
long double 扩展精度 10位有效数字

选择类型的建议:

  1. 明确不可能为负值时,使用unsigned

  2. 一般使用int,short太小不满足需求,long又和int差不多,一旦int不满足,直接使用long long

  3. 算术表达式,不要使用char,bool.因为char在不同的机器上,有无符号.bool则始终是1/0(强转之后)

  4. 浮点运算使用double, float经常精度不满足,同时因为float继续运算的代价与double差不多

不同类型之间进行类型转换时的规则:

  1. integer->bool

    非0为1,其余为0

  2. bool->integer

    只有1/0,所以不能使用bool在算术表达式中

  3. 浮点数->integer

    会丢失精度,去掉小数点后的内容

  4. integer->浮点

    小数点补0

  5. 无符号数 (补码表示)

    对无符号数,赋值出现溢出时,进行取模运算

  6. 有符号数 (补码表示)

    出现溢出,即为未定义行为,具体看编译器实现,UB行为

常用建议: 切勿混用有符号数和无符号数,不然会出现许多意料之外的Bug

二, 变量

1>. 初始化与赋值

在很多程序设计语言中,初始化与赋值,是经常混淆的概念,在C++的某些场合中也是不加以区分的

但是,这两个概念实际上还是有很大区别的

C++中,初始化指的是创建变量,同时赋予其一个初始值.而赋值则是擦出变量当前的值,然后赋以新值
